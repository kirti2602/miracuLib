Alerts

An alert displays a short, important message in a way that attracts the user's attention without interrupting the user's task.

Examples of solid Alerts

Alerts are available in 4 solid variants - Success, Error, Info and Warning. Include class(s) on the basis of whatever Alert(s).

Examples of alphabetical avatars

An avatar can be created using initial letters of user. You need to include class avatar-alpha along with avatar and size class. (e.g. class="avatar avatar-lg-size avatar-alpha").
Don't forget to add flex class to have the alphabets in center(both vertically and horizontally)
