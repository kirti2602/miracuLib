Avatar

Avatar is an object that represents the user's identity on screen.

Examples:-

Avatar is available in 4 different sizes. You can use image in Avatar. You need to include class avatar and for size add class according to size avatar-lg-size, avatar-md-size, avatar-sm-size, avatar-xs-size (e.g. class="avatar avatar-lg-size")

Examples of alphabetical avatars

An avatar can be created using initial letters of user. You need to include class avatar-alpha along with avatar and size class. (e.g. class="avatar avatar-lg-size avatar-alpha")
